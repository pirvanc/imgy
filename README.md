# imgy
Python client for imgy service api

## Getting Started

These instructions will get you a copy of the project for development on your local machine. 


### Prerequisites

Install boto3 library:

```
pip install boto3
```

## Running the tests

Unit test coverage: 30%

Run tests:

```
python test_imgyapiclient.py
```
